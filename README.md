# 🛒 TechVerse

TechVerse is a modern and fully responsive e-commerce frontend web application built using React and Tailwind CSS. It provides a smooth and interactive shopping experience.

---

## 🚀 Features
- 🛍️ Product listing page  
- ❤️ Wishlist functionality  
- 🛒 Add to cart system  
- 💰 Total cart price calculation  
- 📱 Fully responsive design  

---

## 🌐 Live Demo
👉 https://shoptechverse.netlify.app/

---

## 🛠️ Tech Stack
- React.js  
- Tailwind CSS  
- JavaScript (ES6+)  
- Vite
